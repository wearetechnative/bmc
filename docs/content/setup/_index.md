After installing BMC, complete these setup steps to get everything working.

| Step | Description |
|---|---|
| [Shell integration](/setup/shell-integration/) | Required for `bmc profsel` to export `AWS_PROFILE` |
| [Configuration](/setup/configuration/) | `~/.config/bmc/config.json` reference |
| [MFA](/setup/mfa/) | Automatic MFA session refresh with TOTP |

Run `bmc doctor` at any time to check your setup.
